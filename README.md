# at-homework

## Install

```bash
npm install --save at-homework
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'at-homework'
import 'at-homework/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
